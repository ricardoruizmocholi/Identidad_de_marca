## Manual de Marca Interactivo 

Este proyecto es una de mis primeras incursiones en el desarrollo web frontend. Se trata de un **Manual de Identidad Visual** interactivo, diseñado para centralizar y exponer de forma clara los elementos gráficos de una marca.

##  Sobre el Proyecto

El objetivo de este sitio es servir como guía de referencia para el uso correcto de la identidad visual. Fue desarrollado con un enfoque en la legibilidad y la estructura organizada, permitiendo a diseñadores y desarrolladores acceder rápidamente a los recursos de marca.

### Secciones Incluidas:
* **Introducción:** Definición y propósito de la marca.
* **Logotipo:** Directrices sobre el símbolo y la variante secundaria.
* **Colores:** Especificación de la paleta cromática (Principal, Secundarios y Terciarios).
* **Tipografía:** Definición de las familias tipográficas utilizadas.
* **Iconografía:** Estilo visual de los elementos de soporte.

## Tecnologías Utilizadas

* **HTML5:** Estructuración semántica del contenido.
* **CSS3:** Estilización personalizada para mantener la coherencia con la identidad "Blue" y uso de **Flexbox** para el diseño de la cuadrícula de colores y logotipos.

## Aprendizajes Clave

Al ser uno de mis primeros proyectos, me permitió asentar conceptos fundamentales:
1. **Jerarquía Visual:** Cómo organizar la información para que sea fácil de consumir.
2. **Uso de CSS para la Identidad:** Traducir un manual de estilo estático a un entorno web funcional.
3. **Estructura de Carpetas:** Organización básica de activos y archivos de estilo.
